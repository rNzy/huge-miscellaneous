# Composant bux2-message

Ce composant est défini par des attributs :

| Properties      | Type      | Default | Description                                                  |
| :-------------- | :-------- | :------ | :----------------------------------------------------------- |
| data-value      | `string`  | ''      | défini la valeur de l'élément                                |

## Exemple :

```html
<bux2-message>
</bux2-message>
```
