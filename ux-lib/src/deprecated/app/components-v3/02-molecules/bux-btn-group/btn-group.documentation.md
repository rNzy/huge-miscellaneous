# Composant bux2-btn-group

Composant affichant une liste de bouton
Possède le role="group" sur le container, un slot par défaut afin d'accepter les boutons

## Exemple :

```html
<bux2-btn-group>
  <bux-btn></bux-btn>
  <bux-btn>Annuler</bux-btn>
  <bux-btn disabled>Désactiver</bux-btn>
</bux2-btn-group>
```
