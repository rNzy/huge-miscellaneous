# ux-svg

<Meta title="Components|Images-Icones/Icones" />

## Description

Ce composant permet d'afficher des icones. Tous les icones pour une librairie doivent être enregistrés dans le dossier assets/icons au sein de la librairie. Ou importés au sein même du composant si l'icone est spécifique.

## Propriétés

| Name    |           Description           |                   Type                   | Default |
| ------- | :-----------------------------: | :--------------------------------------: | :-----: |
| icon    |         nom de l'icone          |                  String                  |         |
| libSize |        taille de l'icone        | "xs" , "sm" , "md" , "lg" , "xl" , "xxl" |  "md"   |
| circle  |   entoure l'icone d'un cercle   |                 Boolean                  |         |
| rotate  | applique une rotation à l'icone |                  Number                  |         |
| shadow  |   ajoute une ombre à l'icone    |                 Boolean                  |         |
| center  |        centrer l'élément        |                 Boolean                  |         |
| left    |   aligner à gauche l'élément    |                 Boolean                  |         |
| right   |   aligner à droite l'élément    |                 Boolean                  |         |
