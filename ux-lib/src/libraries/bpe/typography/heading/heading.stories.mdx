# ux-heading

<Meta title="Components|Typography/Heading" />

Permet de créer un élément titre pré-stylisé pourvant afficher un titre et un bouton.

## Stories

Par défaut ux-heading crée un span.

### Simple

<Preview withSource="none">
  <Story name="simple" height="60px">
    {stories.simple()}
  </Story>
</Preview>

```html
<ux-heading tag="h2">
  Libellés sur comptes
</ux-heading>
```

### Avec un bouton

<Preview withSource="none">
  <Story name="button" height="60px">
    {stories.button()}
  </Story>
</Preview>

```html
<ux-heading tag="h2">
  Libellés sur comptes
  <ux-btn slot="right" model="primary">Press me</ux-btn>
</ux-heading>
```
