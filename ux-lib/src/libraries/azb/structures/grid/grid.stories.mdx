# ux-grid

<Meta title="Components|Structures/Grid" />

`ux-grid` permet de créer un conteneur possédant entre 2 et 4 enfants, puis de les positionner de plusieurs manières via CSS Grid.

| valeur attribut | nombre de colonnes | répartition en % |
| --------------- | ------------------ | ---------------- |
| 1-1             | 2                  | 50 / 50          |
| 1-2             | 2                  | 33 / 66          |
| 2-1             | 2                  | 66 / 33          |
| 1-3             | 2                  | 25 / 75          |
| 3-1             | 2                  | 75 / 25          |
| 1-1-1           | 3                  | 33 / 33 / 33     |
| 1-1-2           | 3                  | 25 / 25 / 50     |
| 1-2-1           | 3                  | 25 / 50 / 25     |
| 2-1-1           | 3                  | 50 / 25 /25      |
| 1-1-1-1         | 4                  | 25 / 25 / 25 /25 |

**Il est impossible de diviser par plus de 4.**
**Il est impossible de n'avoir qu'une colonne.**
