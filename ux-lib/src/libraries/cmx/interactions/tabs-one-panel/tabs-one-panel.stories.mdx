## Présentation

Ce composant permet de faire des tabs un peu comme l'ux-tabs sont que l'on a
que un panel. Ceci facilite l'intégration dans les framefork js.

A chaque changement de tabs par l'utilisateur (ce qui se fait quand il clique sur la tab ou quand il
la selecitonne par la touche entrer ou espace) il y a deux actions qui sont réalisées :

- l'attribut tab-active du ux-tabs-one-panel change
- le ux-tabs-one-panel emet un custom Event avec :

```js
new CustomEvent('uxTabActive', {
  detail: {
    tab: newTab,
    tabNumber: this.\_getNumberOfTab(newTab)
  },
  bubbles: true
});
```
