# ux-text

<Meta title="Components|Typography/Text" />

Permet de créer un élément html et de le styliser. Utile pour éviter de créer des classes utilitaires.

## Stories

Par défaut ux-text crée un span.

### Span

<Preview withSource="none">
  <Story name="span" height="40px">

  </Story>
</Preview>

```html

```

### Gras

#### `<strong>` versus span en gras

Si l'on veut rendre un texte en gras sans vocaliser son emphase (par exemple vocalisation d'un chiffre ou d'un détail), on utilise `bold`.

<Preview withSource="none">
  <Story name="bold" height="40px">
    {stories.bold()}
  </Story>
</Preview>

```html
<ux-text bold></ux-text>
```

Dans un cas où l'on veut marquer l'emphase (notamment lors de la vocalisation), préférer le html sémantique avec `<strong>`.

<Preview withSource="none">
  <Story name="strong" height="40px">
    {stories.strong()}
  </Story>
</Preview>

```html
<ux-text tag="strong"></ux-text>
```

Le résultat visuel est le même mais **la sémantique est importante pour l'accessibilité** !
