# Git

## Merge Request

en ligne de commande dans le terminal:

1. s'assurer qu'on est sur la branche develop et que celle ci soit à jour: `git pull`
2. créer sa feature branch ( `git branch <mabranche>̀` ), puis se déplacer sur la branche nouvellement créée ( `git checkout <mabranche>` )
3. effectuer ses modifs puis:

```bash
git add <mesfichiers>
git commit -m 'mon joli msg'
git push -u origin <mabranche>
```

4. et après tu peux faire ta PR sur gitlab en suivant le lien proposé dans le terminal
