# Installation

> Le projet necessite la version de node >=8.11.4 (il marche avec la 10.15)

Pour vérifier la version utilisée

```bash
node -v
```

> Pour installer et changer de version de node, il est possible d'utiliser l'utilitaire [NVM](https://github.com/creationix/nvm/blob/master/README.md)

Cloner le projet puis l'installer

```bash
git clone https://gitlabtux2-a.l.infra.arkea.com/BAD/9020/ux-library.git ux-library
cd ux-library
npm install
```

## Démarrer un serveur local

```bash
npm run dev:EFS
```

Où EFS vaut le projet sur lequel vous voulez travailler

### Documentation storybook

```bash
npm run storybook:EFS
```

Où EFS vaut le projet sur lequel vous voulez travailler

Ceci lance un serveur, par exemple pour le cmb sur le port 7001 : http://localhost:7001/

L'adresse du serveur est affiché dans le terminal.
