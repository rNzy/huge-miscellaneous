# Generalitées

La classe de base expose des méthodes pour facilité le dev :

`this.$;` => Permet d'accéder au shadowroot.

`this.uxLibUrl;` => Chemin d'accès à l'uxLibUrl.

`this.container;` => Élément dans le template avec la classe `.js-container` ou sinon this.

`this._slots;` => Tableau des slots.

`this.setBooleanClass;` =>

```js
/**
 * Fonction qui permet d'ajouter ou supprimer une classe à un élément
 * en fonction de la valeur d'un boolean
 * @param {boolean} boolean  put/remove
 * @param {String} className the class name
 * @param {HTMLElement} el put/remove class on this element (optional default is this.container)
 */
```

`this.navigator;` =>

```js
/**
 * Get navigator with version
 *
 * output exemple :
 * {
 *  name: 'Chrome',
 *  version: '62'
 * }
 */
```
