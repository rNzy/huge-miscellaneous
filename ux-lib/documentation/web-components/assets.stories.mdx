# CSS Guide Line

## Insertion d'image :

Dans le repertoire du composant mettre le fichier image.
Prenons l'exemple de test.png.

Les images sont souvent spécifiques à un composant donc elle doivent se trouver dans
le repertoire de celui-ci.

Prenons l'exemple suivant :

```
  src
    libraries
      efsexemple
        formulaires
          buttons
            - button.component.js
            - ...
            - monimage.png
```

Il faut choisir la méthode css ou js mais pas les deux.

### Pour inserer l'image depuis le js

Dans le fichier button.component.js du composant écrire :

```js
// On l'importe pour que webpack la traite
import monimage from './monimage.png';

...

// On l'utilise ou on veut
// NB le this.uxLibUrl est automatiquement fournis par la base (cf. src/core/base/base.mixin.js)
return `<img src="${this.uxLibUrl}/${monimage}" />`;
```

### Pour insérer l'image depuis le css

Dans le fichier button.styles.scss :

```css
.maclasse {
  background: url('./monimage.png');
}
```

## Cas spécifiques (image globale non spécifique à un seul composant) :

La première chose est de se poser la question. Cette image est-elle
spéficique ou non ? Dans la plupart des cas la réponse est : elle est spécifique.
Mais si vous êtes sur qu'elle doit etre globale alors vous pouvez lire la suite.
